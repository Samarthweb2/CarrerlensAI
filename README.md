# CareerLensAI

CareerLensAI is an intelligent Student Resume Analyzer designed to parse, analyze, and extract insights from student resumes. It helps match candidates with relevant job descriptions, identify skill gaps, and provide actionable feedback.

## Project Structure

```text
CareerLensAI/
├── docs/           # Documentation and design assets
├── backend/        # Backend server, APIs, and business logic
├── frontend/       # Frontend user interface (web/desktop)
├── database/       # Database schemas, migrations, and seed scripts
├── data/           # Sample data, training datasets, and resumes
├── dashboards/     # Analytical dashboards and reporting tools
└── README.md       # Project overview (this file)
```

## Features

- **Resume Parsing**: Extract key information (education, experience, skills, projects).
- **Skill Gap Analysis**: Compare resume skills against target job roles.
- **Career Path Recommendations**: Suggest potential job matches and learning resources.
- **Visual Analytics**: Interactive dashboards for recruiters and students.
